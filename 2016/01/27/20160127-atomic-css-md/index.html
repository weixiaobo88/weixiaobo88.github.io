什么是OOCSS

OOCSS 是 Object Oriented CSS

  • structure 和 skin 分开: 定义一个 ‘gradient’, 然后在其他地方 extend 这个 gradient
  • container 和 content 分开: 不要级联选择器, 直接写在一个class中, 然后引用这个 class

更多资料:

  • Official OCSS documentation on GitHub
  • Nice read about OCSS in Smashing Magazine

什么是Atomic Approach

Atomic Approach, 出自 Atomic Design, 因为文章Challenging CSS Best Practices而得到普及.

同时 Atomic Approach 可以用来组织项目的文件, 参考 The “Other” Interface: Atomic Design With Sass

Atomic Design :

  • Atoms
  • Molecules
  • Organisms
  • Templates
  • Pages

什么是ACSS

每个选择器就定义一次, 然后在 markup 里引入对应的 class name

缺点:

  • 不能用 media queries, 因为现在样式写在了 markup 上, 不能通过 media queries 来改变
