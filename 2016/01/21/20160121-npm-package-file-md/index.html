package.json文件

main field

main是程序的主要入口点, 是一个 module 的 ID.

  • 默认情况下是相对于包所在文件夹的根目录.
  • 也有可能是作为 main script.

举个例子, 如果你的包名为 foo, 当用户安装这个包之后, 可以使用 require('foo').
