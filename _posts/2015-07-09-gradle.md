Task Dependency

dependencies {
  compile 编译源代码时需要的
  testCompile 比如junit
  runtime 比如servlet，对编译没有影响，在web容器启动后才需要
  testRuntime 
}

task mkdir << {
     file("./src/main/java").mkdirs();

}


Ant maven gradle区别
ant: 构建操作小的低级的封装，无法感知到war和第三方的依赖；没有指定的目录结构
     target: copy,compile,jar
maven: 
     目录结构规范化了，src/main/java
     引入了dependency
     内建项目的周期管理，扩展性不好
gradle:
     maven基础上，task之间有依赖关系，
